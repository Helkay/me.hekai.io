idea 通过跳板机 远程debug

工具 idea 2019/10/18

首先在 java 启动的时候加上参数: 指定一个端口 (以下为5066 )

然后通过远程建立ssh通道将本地的5066,代理到服务机的5066端口上:

  • JUMP_SERVER_USER : 是跳板机账号
  • SERVER_IP : 是服务器的ip，跳板机通过该ip可连接到服务器
  • JUMP_SERVER_PORT : 是指跳板机的ssh端口
  • JUMP_SERVER_IP : 是指跳板机的ip

然后idea上配置远程debug后启动即可
